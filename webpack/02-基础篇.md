# 一、核心概念——Entry
> Entry用来指定webpack的打包入口。


![image](https://user-images.githubusercontent.com/72426886/132118607-c71c63e9-11d7-403e-b8da-365de1c75490.png)
依赖图的入口是entry，对于非代码、字体依赖也会不断加入到依赖图中。
## 1.1单入口: entry是一个字符串
```js
module.exports = {
  entry: './src/index.js'
}
```
## 1.2多入口：entry是一个对象
```js
module.exports = {
  entry: {
    app: './src/app.js',
    adminApp: './src/adminApp.js'
  }
}
```
